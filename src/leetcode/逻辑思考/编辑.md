## leetcode72 编辑距离

给你两个单词 word1 和 word2， 请返回将 word1 转换成 word2 所使用的最少操作数  。

你可以对一个单词进行如下三种操作：

* 插入一个字符
* 删除一个字符
* 替换一个字符

```java
class Solution {
    public int minDistance(String word1, String word2) {
        int n = word1.length();
        int m = word2.length();
        // dp[n][m]理解为，word1前n个字符和word2前n个字符的最小编辑距离
        int[][] dp = new int[n+1][m+1];
        // 其中一个长度为0时，编辑距离就是另一个字符的长度
        for(int i =0; i <= n; i ++){
            dp[i][0] = i;
        }
        for(int i =0; i <= m; i ++){
            dp[0][i] = i;
        }
        for(int i = 1; i <= n; i ++){
            for(int j  =1; j <= m ; j ++){
                char w1 = word1.charAt(i - 1);
                char w2 = word2.charAt(j - 1);
                if(w1 == w2){
                    dp[i][j] = dp[i-1][j-1];
                }else {
                    // 三个方向都有可能编辑
                    dp[i][j] = Math.min(dp[i-1][j-1],Math.min(dp[i-1][j],dp[i][j-1])) + 1;
                }
            }
        }
        return dp[n][m];
    }
}
```

## 一次编辑

```java
class Solution {
    public boolean oneEditAway(String first, String second) {
        int leng1 = first.length();
        int leng2 = second.length();
        if(Math.abs(leng2- leng1) > 1){
            return false;
        }
        int l = 0;
        // 短的字符串
        int r = Math.min(leng2,leng1);
        int r2 = Math.max(leng2,leng1);
        int dl = 1;
        // 左边相同字符串长度
        while(l < r ){
            if(first.charAt(l) == second.charAt(l)){
                l ++;
            }else {
                break;
            }
        }
        // 右边相同字符串相同
        while(dl <= r){
            if(first.charAt(leng1 - dl) == second.charAt(leng2 - dl)) {
                dl ++;
            }else {
                break;
            }
        }
        // 左右长度相加不能小于 （总长度-1）
        if(r2 - l - dl + 1 > 1){
            return false;
        }
        return true;
    }
}
```
